# Kumar Store Sales Dashboard 2022

## 📊 Project Overview
This project focuses on analyzing sales data of Kumar Store for the year 2022 using Microsoft Excel.  
An interactive dashboard was created to understand sales performance, customer behavior, and order trends.

---

## 🎯 Objective
- Analyze sales data to identify trends and patterns  
- Understand customer demographics and preferences  
- Evaluate performance across different states and sales channels  

---

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- Data Cleaning  

---

## 📈 Key Insights
- Women contribute approximately 65% of total sales  
- Maharashtra, Karnataka, and Uttar Pradesh are the top-performing states  
- Amazon, Flipkart, and Myntra contribute the majority of total sales  
- Most orders are successfully delivered, indicating good order fulfillment  

---

## 📌 Dashboard Features
- Interactive filters using slicers (Month, Category, Channel)  
- Sales vs Orders trend analysis  
- Sales distribution by gender and age group  
- Top 5 states by sales  
- Order status breakdown
